
# 概要
基于pyqt6 教务管理系统 + Mysql ，需要帮忙部署可以联系我!



# 项目需求

```
登录角色：管理员、老师、学生

1、管理员可以创建/编辑课程，添加/编辑老师，冻结学生，给老师发工资，以及査看流水,设置虚拟学习人数，以及设置主页的欢迎语

2、管理员可以査看KPI数据，包含累计付费人次、今日营收、累计营收、累计注册学员、总课程数量、老师人数、课程包含价格、学习人数、授课老师

3、学员注册后，可以随意购买课程，购买课程之后，可以查看授课老师的联系方式

4、老师只能由管理员创建，并且只有当管理员给课程设置授课老师之后，该老师才可以看到对应课程， 并且安排已购课学员学习，老师登录之后，可以设置自己的联系方式

5、学员和老师可以查看热销课程数据

6、所有功能必须登录之后才能使用
```



# 1、需求分析

```
管理员
	1、登录功能
	2、创建课程功能
	3、编辑课程功能
	4、添加老师功能
	5、编辑老师功能
	6、冻结学生功能
	7、给老师发工资
	8、查看流水
	9、设置虚拟学习人数
	10、设置主页的欢迎语
	11、查看KPI数据
	
学生
	1、注册功能
	2、登录功能
	3、购买课程功能
	4、查看已购买的课程以及教师的联系方式
	
老师
    1、登录功能
    2、管理课程功能(安排学生学习)
    3、设置联系方式功能
    4、查看热销课程数据
    5、退出账号功能
```



# 2、架构设计

```
基于MVC模式

-用户视图层:与用户交互
	-core
    	login_register_view.py 
        	
-逻辑接口层:做核心的业务逻辑处理
	-interface
		-user_interface.py -一些公共的接口
		--
		
-数据处理层:用于做数据的增删改查 使用Mysql 8.0
	-db
    	-models.py -模版
    	-db_handler -数据库操作程序
    		-db_common.py - 公共的数据库操作方法
    		--
    	
-配置文件
	-conf
		-settings.py

-公共方法
	-lib
		-common.py

-界面文件
	-gui
		-login.py

-日志文件
	-log
		-user.log
	
-start.py -入口文件
-conf.init -配置数据库参数
--demo.sql -数据库表搭建
```
